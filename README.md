# 💙 Living Brilliantly with Type 1 Diabetes

A warm, empowering family companion app for navigating a T1D diagnosis together.

## What's inside

- **For Kids** — Age-appropriate modules to help children understand T1D and feel empowered
- **For Parents** — Evidence-based guides covering the science, emotional support, school navigation, and technology
- **Inspiring Lives** — Real people thriving with T1D across music, sport, law, and more
- **Resources** — Direct links to Breakthrough T1D, Beyond Type 1, Diabetes Australia, and more

---

## Running locally

Make sure you have [Node.js](https://nodejs.org) installed (version 18+), then:

```bash
npm install
npm run dev
```

Open your browser at `http://localhost:5173`

## Deploying to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"**
4. Select your repository
5. Vercel auto-detects Vite — just click **Deploy**
6. Your app will be live at a `.vercel.app` URL instantly ✨

No configuration needed — Vercel handles everything automatically.
